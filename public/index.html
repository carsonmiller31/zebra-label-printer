Zebra GX420d Label Designer

Design custom labels and print them to your printer over the network.

Printer Connection

GX420d uses raw port 9100. Hold the feed button to print a config label showing the IP.
"Restore Printer Defaults" reloads the printer's saved settings (^JUR) — use it if other software starts printing off after using this app.
